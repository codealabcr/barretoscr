# Fotos del cliente (intake)

Aquí van las fotos crudas que el cliente envía por WhatsApp/correo, antes de procesarlas para el sitio.

**Flujo:**
1. Guardar la foto original aquí con un nombre descriptivo (p. ej. `botas-justin-vitrina.jpg`).
2. Al integrarla al sitio: optimizar (resize + compress), mover a `assets/` y referenciar desde `index.html`, reemplazando el placeholder de Unsplash correspondiente y completando el `<!-- TODO alt: ... -->`.
3. No borrar los originales de esta carpeta — quedan como respaldo del material entregado.
